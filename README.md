# BLOOM

**BLOOM** is an educational and interactive mobile application designed to support children with Down syndrome in learning daily routines and basic academic skills in a simple, engaging, and accessible way.  
The app focuses on building **independence**, **confidence**, and **cognitive development** through guided visual and audio experiences.
